Google's Firebase

Firebase is Google's web and mobile app development platform that you can use to deploy sites and web apps. Firebase offers cloud services and tools to help developers build and deploy projects without needing to worry about having their own infrastructure.

Firebase, I deployed a website within Firebase using Firestore and authentication in order to gain access. Firebase makes it very easy to create webpages and webpages using their built in tools.
