Amazon Web Services

AWS is Amazons cloud computing solution. Using AWS, you can run applications and webpages on cloud servers, host your files securely on the cloud, and build managed databases. AWS offers an integrated development environment for developers, and plenty of other tools to help you build, test, and publish your code.

Cloud9

Using AWS, I created a website within Cloud9, and deployed it into a bucket within Amazon S3. This consisted of creating an Amazon AWS EC2 instance, setting up my live server, setting up my S3 bucket, and then setting my access policies. After the setup, I developed my webpage using cloud9, and dropped my files into the bucket.
